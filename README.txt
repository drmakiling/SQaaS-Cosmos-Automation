
In order to run all of the tests with pytest, run the following commands:
    cd tests (if not in the tests folder already)
    pytest

If you want to see the automation running, add the --headed flag 

In order to run all of the tests with behave, you need to be in the main folder run the following command:
    behave

If you want to see a specific tag, add the --tags=@tag_Name flag with tag_Name set to the tag that you want to run
